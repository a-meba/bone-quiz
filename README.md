# 🦴 BONE STRUCTURE QUIZ 🚀 🎲 🔥

본 프로젝트는 간호/보건/체육 계열 학생들을 위한 **인체 골격(Skeletal System) 명칭 주관식 학습 시스템**입니다. 

복잡한 인체 해부학 이미지를 보며 직접 명칭을 입력함으로써, 단순 암기를 넘어 실제 위치와 이름을 매칭하는 고효율 학습을 지향합니다.

---

## 🔗 주요 정보
* **배포 URL** : [https://a-meba.github.io/bone-quiz/)
* **최신 버전** : `v1.2.0 STABILITY UPDATE`
* **지원 기기** : PC, 태블릿 (가로 모드 권장), 모바일

## 📖 프로젝트 소개
인체 골격의 기초가 되는 **주요 부위의 명칭(신용어, 구용어, 영어)** 을 마스터하기 위해 제작되었습니다.  //신용어,구용어 전부 반영x

이미지 위의 빈칸을 클릭하여 직접 입력하는  주관식 방식을 채택하여 해부학적 위치 감각과 철자 암기력을 극대화합니다.

## ✨ 주요 기능과 특징
* **시각적 인터랙티브 학습**
  * 실제 인체 골격도 위에 배치된 라벨을 클릭하여 직관적으로 퀴즈를 풀 수 있습니다.
* **다국어/다양한 정답 지원**
  * 신용어(예: 빗장뼈), 구용어(예: 쇄골), 영어(예: clavicle)를 모두 인식하는 스마트 채점. //일부 미지원 
  * 공백 및 대소문자 예외 처리를 통한 유연한 정답 체크.
* **실시간 결과 표시**
  * 입력 즉시 정답 여부와 함께 정확한 풀 네임(한글+영어)을 표시하여 즉각적인 피드백을 제공합니다.
