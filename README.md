# 🦴 BONE STRUCTURE QUIZ 🚀 🎲 🔥

본 프로젝트는 간호/보건/체육 계열 학생들을 위한 **인체 골격(Skeletal System) 명칭 주관식 학습 시스템**입니다. 

책만 보고 외우다간 머리 터질 것 같아서, 실제 해부학 이미지를 보며 빈칸에 이름을 직접 때려 박는 **고효율 암기 메타**를 지향하며 만들었습니다.

---

## 🔗 주요 정보
* **배포 URL** : [https://a-meba.github.io/bone-quiz/)
* **최신 버전** : `v1.2.0 STABILITY UPDATE`
* **지원 기기** : PC나 태블릿 가로 모드가 제일 보기 편합니다. (폰으로도 틈틈이 가능!)

## 📖 프로젝트 소개
인체 골격의 기초가 되는 **주요 부위의 명칭(신용어, 구용어, 영어)** 을 마스터하기 위해 제작되었습니다.
> ⚠️ **솔직 고백:** 신용어, 구용어 전부 다 넣으려다 기절할 뻔해서... (전부 반영은 X)

이미지 위 빈칸을 클릭해서 직접 입력하는 주관식 방식이라, 위치 감각이랑 스펠링 암기 하나는 확실하게 됩니다.

## ✨ 이런 기능이 있어요
* **직관적인 클릭 퀴즈**
    * 실제 뼈 사진 위에 있는 라벨(이라 쓰고 **노가다 상자**라 읽습니다)을 클릭해서 바로바로 풀면 됩니다.
* **유연한 정답 처리**
    * 빗장뼈(신), 쇄골(구), Clavicle(영)을 다 알아먹으려고 노력하는 스마트 채점 시스템!
    * *(일부 미지원 단어가 있을 수 있지만)* 웬만한 공백이나 대소문자는 알아서 다 걸러줍니다.
* **실시간 결과 표시**
    * 엔터 치자마자 정답인지 알려주고, 정확한 풀 네임(한글+영어)을 표시해줘서 즉각적인 피드백을 제공합니다.
